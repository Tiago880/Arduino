## 🔖&nbsp; Sobre
Nesse projeto como usar o keypad 4x4 no arduino usando também o display  LCD 16x02 . A ideia é fazer o display mostrar o que foi digitado no keypad invés de mostrar no monitor serial da IDE do Arduino.
